### 복소수

```python
# i대신 j를 쓴다, 대문자도 가능
g=8+7j

h=g.imag
# .imag는 복소수 부분 반환 (여기서는 7.0)

i=g.conjugate()
# .conjugate()는 켤레복소수 반환 (여기서는 8-7j)
```

# 튜플

리스트와의 차이점
 
* 리스트 : 항몫값 변화 가능
* 튜플 : 항몫값 변화 불가능

튜플은 삭제, 변경은 안되고 / 인덱싱, 슬라이싱([1:4])은 가능, 더하기 곱하기도 가능(붙이기,반복)
